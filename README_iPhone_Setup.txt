CAIB 2 iPhone PWA Setup

This PWA includes flashcards, MCQ, and fill-in-the-blank practice. Progress is stored locally on your device.

Best iPhone method:
1. Upload this whole folder to Netlify Drop, GitHub Pages, or any HTTPS web hosting.
2. Open the HTTPS link in iPhone Safari.
3. Tap Share → Add to Home Screen.
4. Open from the new icon. It will work offline after first load.

Important: iPhone cannot properly install a PWA from a local Files app HTML file. It needs HTTPS for Add to Home Screen + offline service worker.
